“亲，我下载了一款清单软件，就是只能试用一个月”

## 清单软件

我一看老婆给我发的清单软件截图，就觉得好乱好 low。

**敬业签**，我查了下，是这个软件。我就想着，一个简单的任务清单，应该又很多免费平替吧。

网上一搜，要么国外的，老婆下载不来。要么国内的，收费、会员、老古董。

**滴答清单**，这是我在 v2 中搜到的，但是面板好大，内容多，诱导付费。

## 空白

我发现，真没有一款值得推荐给老婆使用的清单软件。那就自己开发吧。

两个需求：
* 任务清单
* 可以缩小

## 开发

Clone Electron 模板，删除不用的东西。开发极简功能。调整 UI 样式，设计 icon。

### 打包

打包反而耽误不少时间、一会权限、一会网络、一会 icon、一会功能。。。

## 分享

虽然功能小，我感觉还是有人需要。
